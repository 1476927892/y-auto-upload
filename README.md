# y-auto-upload

## 安装

```
npm install y-auto-upload -D
```

## 配置

```
在项目根目录创建 yau.json
{
  "server": {
    "host": "",
    "port": "",
    "username": "",
    "password": ""
  },
  "address": {
    "path": "", //服务器目录  /usr/local/nginx/html
    "projectName": "",  //服务器存放项目的文件夹
    "backupName": ""  //备份文件夹
  }
}

```
